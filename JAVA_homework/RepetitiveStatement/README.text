맞출때까지 계속하는 숫자 야구게임을 만들어보자!
중복없는 랜덤한 세자리 숫자를 생성한다.
자리와 숫자가 일치하면 스트라이크 +1
자리는 다르지만 숫자가 일치하면 볼 +1
일치하는 숫자가 없으면 아웃 +1
(정답: 6 8 4)
세 자리의 숫자 입력 >> 456
1차 시도(456) : 0S 2B 1O
----------------------------
세 자리의 숫자 입력 >> 687
2차 시도(687) : 2S 0B 1O
----------------------------
세 자리의 숫자 입력 >> 684
3차 시도(684) : 3S 0B 0O
----------------------------
